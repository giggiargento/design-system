Introduction

Get started with Panelita, the health most popular framework for building responsive, mobile-first sites, with css and a template starter page.

Quick start
Get started by downloading the compiled CSS and JS in Download section, then:
CSS
Copy-paste the stylesheet <link> into your <head>before other stylesheets to load our CSS. 
<link rel="stylesheet" href="https://...css" >
content_copy
JS
Many of our components require the use of JavaScript to function.
Place one of the following <script></script> near the end of your html, right before the closing <body> tag, to enable them. 
<link rel="stylesheet" href="https://...css" >
content_copy
Notifications
To use our notifications system first Download dependencies, then place the following <script></script> near the end of your html, right before the closing <body> tag, to enable them. 
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
<script src="./notifications/bootstrap-notify.min.js"></script>
<script src="./notifications/bukeala-notify.js"></script>
