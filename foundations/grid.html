Grid

The grid is the foundation for positioning elements onscreen. Designing to the grid helps create seamless, easy to follow experiences.

To facilitate many different screen sizes, we use a 4 pixel grid system that expands the space between columns by multiples of 4 pixels. This helps in creating responsive pages from elements with fixed sizes that work on many devices.
